# MadBird Studio

A premium, Awwwards-caliber marketing agency website built with Next.js, React, Three.js, GSAP, and Lenis.

## Features

- **Next.js 16 (App Router)** - Fast, modern React framework
- **Three.js & React Three Fiber** - 3D interactive hero section
- **GSAP** - Complex, high-performance animations
- **Lenis** - Smooth scrolling experience
- **Tailwind CSS** - Utility-first styling with custom tokens
- **Fully Responsive** - Flawless across all devices

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
